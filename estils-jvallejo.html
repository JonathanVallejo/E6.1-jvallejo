Estilos

Etiquetas

Las etiquetas son palabras que componen el lenguaje HTML. Permiten definir la estructura y el funcionamiento de cualquier página web. Las etiquetas deben escribirse entre los símbolos
"<>", para poder distinguirlas del texto normal de la página.

Las etiquetas se deben escribir por pares, una de apertura y otra de cierre. En algunos casos basta con el de apertura, pero es una buena norma de programación indicar siempre ambas etiquetas.
La de cierre se distingue porque comienza con el símbolo "</".

A estas etiquetas se le pueden añadir una serie de parámetros, que permiten cambiar el color de fondo de la página, de los enlaces, etc.

En algunos casos es necesario aportar datos adicionales. Por ejemplo, cuando se define un Link hay que especificar su destino.

Los parámetros deben ir siempre entre los símbolos " < " y " > " para que sean tenidos en cuenta por el navegador.

ID

El atributo ID se utiliza para definir una zona específica de una página (el encabezado, pie de página, algun menú).

Se utiliza un ID si esa etiqueta es única, no debe haber más de una etiqueta con el mismo ID en nuestra página.

Para poder utilizar este atributo debemos definir sus propiedades escriviendo el símbolo # antes del nombre.

Clases

Este atributo se utiliza para definir estilos generales que podemos personalizar, como por ejemplo en un párrafo o un enlace.

Al contrario que el atributo ID, este si podemos poner tantos como queramos, y lo utilizaremos cuando queramos un mismo estilo para varias zonas de nuestra página.

Para definir los parámetros o propiedades de una clase lo escriviremos con un punto "." antes del nombre.
