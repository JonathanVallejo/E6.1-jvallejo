Tabla CSS

Atributo Descripción Explicación Ejemplo
background-color La propiedad background-color establece el color de fondo de un elemento. Como parámetro se pone el color en formato hex, rgb o rgba. background-color: #FFF;
background-color: #FFFFFF;
background-color: rgba(255,134,56,0.8);
background-image Sirve para agregar varias imágenes como una sola. Su parámetro es url() background-image:url(pokemon.svg)
background-repeat Sirve para que se repita una imágen tanto vertical como horizontalmente,
funciona junto a background-image.		 Sus parámetros son repeat, no-repeat, repeat-x, repeat-y. background-repeat: repeat-x;
font-size Nos permite determinar el tamaño del texto Algunos de sus parámetros son: medium, large, small, %. font-size:150%;
font-weight Nos permite definir la anchura del texto Los parámetros de este son: normal, bold, lighter o number. font-weight: 500;
font-family Nos permite determinar la fuente del elemento En este caso el parámetro sera el nombre de la fuente. font-family: verdana;
text-align Sirve para determinar la posicion del texto. Los parámetros son: left, right, center o justify. text-align: right;
text-tranform Sirve para cambiar el texto entre mayúsculas o minúsculas, según el
parámetro que le indiquemos.		 Los parámetros son: none, capitalize, uppercase o lowercase. text-tranform: uppercase;
border Se utiliza para modificar los bordes de un elemento. Los parametros que se utilizan son: style, color, width, top, left,
right o bottom.		 border-color: red;
list-style-type Se utiliza para listar el conenido con algún estilo. Sus parametros son disc, armenian, circle, lower, georgia, upper,
decimal o none.		 list-style-type: disc;
text-decoration Nos permite añadir decoraciones al texto. Sus parametros son underline, overline, line-through o none. text-decoration: underline;
padding Sirve para deterinar el espacio. sus parámetros son: bottom, left, top, right. Padding-right: 40px;
margin Este atributo támbien nos permite definir espacios sus parámetros son: bottom, left, top, right. margin-right: 40px;.
